# Yukiss Store

一个手帐品牌的 D2C 电商 demo，用于验证 AI 在电商全流程中的介入价值和体验提效。

## 项目定位

Yukiss 不是一个真实产品，而是一个有完整业务叙事的实验场——通过还原一个手帐独立站的完整购物流程，系统地探索 AI 在每一个环节能做什么、不能做什么。

## 业务流程

| 层级 | 内容 | 状态 |
|------|------|------|
| v0.1 Discovery Layer | 搜索筛选 + AI 自然语言导购 | 🚧 进行中 |
| v0.2 PDP Layer | 产品详情 + AI 推荐 | 📋 待开始 |
| v0.3 Checkout Layer | 结账流程 + AI 风控 | 📋 待开始 |
| v0.4 Post-Purchase Layer | 售后 + AI 客服 | 📋 待开始 |
| v0.5 Account Layer | 账户记录 + AI 个性化 | 📋 待开始 |
