## 项目简介

Yukiss 是一个手帐品牌的 D2C 电商 demo，用于验证 AI 在电商全流程中的介入价值和体验提效。

完整流程覆盖：搜索发现 → 产品详情 → 结账 → 售后 → 账户记录。

## v0.1 Discovery Layer

### 这一层在做什么

用户进入 Yukiss 后的第一个场景——浏览和搜索商品，找到感兴趣的产品。

### 核心用户流程

1. 用户进入首页，浏览商品列表
2. 用户输入自然语言需求
3. AI 将需求转化为结构化 filter 选项
4. 用户基于 filter 结果浏览，进入产品详情页

### AI 介入点

**自然语言 → Filter 转化**：用户输入"我想找适合子弹笔记的本子，封面要硬一点"，AI 自动解析并映射为结构化筛选条件：

- 用途：子弹笔记
- 装订：硬壳封面
- 规格：（待定）

同时支持多轮对话：用户可通过多轮对话逐步缩小搜索范围，AI 保留上下文并支持相对表达（如“便宜一点”表示在当前价格基础上降低，“换个颜色”表示切换当前筛选的颜色属性）。

### 待确认的设计决策

- [ ] 自然语言转化失败时的 fallback 策略
- [x] 是否支持多轮对话式筛选（已实现：支持多轮对话，每次对话保留上下文，支持相对表达如“便宜一点”、“换个颜色”）
